# Codes-for-Post-selected-MIPT
The repository contains the codes to compute various entanglement measures in a post-selected circuit and a random tensor network. The repository has 2 folders:

- `codes for random quantum circuit` - The codes for the post-selected random quantum circuit.
- `RTN_code_paper` - The codes for the random tensor network.

Each of these folder have their own `README.md` files on how to run these codes.
